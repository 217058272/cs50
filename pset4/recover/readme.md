[Pset 4 recover](http://docs.cs50.net/problems/recover/recover.html)

## How to run
0. `cd pset4/recover`
1. `make recover`
2. `./recover card.raw`

## What it does
Assembles jpeg images by leveraging the fact that first 4 bytes are always 0xff 0xd8 0xff 0xe0-0xef
